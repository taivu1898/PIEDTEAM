## Variable - Biến

Khai báo biến trong JavaScript

- Cách 1: var - xuất hiện từ phiên bản ES đầu tiên

```js
let msg = "Hello World";
```

- Cách 2: let và const - xuất hiện từ phiên bản ES6

```js
let a = 10;
var b = 11;
```

Const và Object

```js
const profile = {
  name: "Oggy",
  height: 160,
};
```
